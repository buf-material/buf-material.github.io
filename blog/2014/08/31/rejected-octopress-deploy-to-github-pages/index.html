OctopressのコンテンツデータをGitHub Pagesへdeployしようとしたらrejectedで失敗する件

Aug 31st, 2014

OctopressをGitHub pagesでホスティングさせているのですが、GitHubレポジトリへのdeployタスクがrejectedとなって失敗してしまうことがあります。

暫定対処

他にも解決方法があるかもしれませんので暫定と思っています。

OctopressのRakefileに記述されているdeployタスク内で実行されているpushタスクをforceオプションで実行します。

git pushコマンドのremoteレポジトリ名の頭に+を付加することでforceオプションと同じ効果を付加できるようです。
